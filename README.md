# PMMP-Plugins
PMMPのプラグインの一覧  
ライセンスなんて無かったんや…(錯覚)  
  
一部のプラグインはコードの変更をしてるよ(ちょっと)  
例: パフォーマンスの改善、解凍禁止コードの削除… など

もし動かないプラグインがあったらissueとかTwitterで教えてね
